School Management API

Welcome to the School Management API. This API provides endpoints for managing school data with location-based sorting capabilities.

API Endpoints

POST /api/addSchool

Add a new school with name, address, and geographical coordinates.

Request Body:

{ "name": "School Name", "address": "School Address", "latitude": 34.0522, "longitude": -118.2437 }
GET /api/listSchools?latitude=34.0522&longitude=-118.2437

List schools sorted by proximity to the provided coordinates.

GET /api/test

Test if the API is working properly.

GET /api/health

Check the health status of the API.

Technical Details

This API is built with Node.js and Express, using SQLite for local development and in-memory storage for serverless deployment.

The distance calculation uses the Haversine formula for accurate geographical distance measurement.
